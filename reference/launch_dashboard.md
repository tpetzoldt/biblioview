# Launch the Biblioview Dashboard Locally

Launch the Biblioview Dashboard Locally

## Usage

``` r
launch_dashboard(group = NULL, key = NULL, folder = NULL, title = NULL)
```

## Arguments

- group:

  Optional character string containing a default Zotero Group ID.

- key:

  Optional character string containing a default Zotero API Key.

- folder:

  Optional character string containing a sub-collection (folder) of the
  group library.

- title:

  Optional character string to override the default portal header title.
