# biblioview: A Shiny Web Portal and Abstract Enricher for Zotero Libraries

Provides automated pipelines to fetch Zotero group libraries, generate
APA citations, and harvest citation counts and missing abstracts from
external metadata APIs (Crossref, OpenAlex, and Europe PMC).
